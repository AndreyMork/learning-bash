| Command | Meaning |
|---|---|
| `cat` | Concatenate files |
| `sort` | Sort lines of text |
| `uniq` | Report or omit repeated lines |
| `grep` | Print lines matching a pattern (`-i` ignore case, `-v` do not match the pattern) |
| `wc` | Print newline, word, and byte counts for each file |
| `head` | Output the first part of a file (`-n` number of lines, `-f` realtime) |
| `tail` | Output the last part of a file (`-n` number of lines) |
| `tee` | Read from standard input and write to standard output and files |
